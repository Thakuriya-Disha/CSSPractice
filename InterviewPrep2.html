What is box-sizing?

Box-sizing defines how width and height are applied to an element. There are 2 types of values of the box-sizing property: 1) content-box and 2) border-box. The default value is: content-box.
  • content-box: The actual width is the sum of content-width, padding and border.
    Changing padding and border changes the actual width of the element.
  • border-box: The actual width includes all three: content-width, padding and border.
    Changing padding and border does not change the actual width of the element.
That is why despite both blue and pink boxes having the same values for height and width (i.e. 200px), the boxes appear different.

In an animation, an element gradually changes from one style to another. To implement an animation, keyframes are used. Keyframes define animation rules and give us more control over animation.
An element goes from initial state to final state with intermediate steps in between. And animations do not need a trigger.

What is grid-template-columns used for?

Grid is a 2-dimensional property, that arranges contents in both rows and columns. It is useful to make complex patterns/structures.

grid-template-columns is used to create columns in a grid. It specifies the number of columns and the width of those columns.

Which property will you use to merge cells vertically & horizontally in a grid?

To merge cells vertically, we use: grid-column-start and grid-column-end

To merge cells horizontally, we use: grid-row-start and grid-row-end.

What is the difference between display:none and visibility:hidden?

When display:none is used, the element is removed altogether. No empty space is left
In visibility:hidden, it hides the element and an empty space is left behind. This is preferred as the overall structure remains unaffected.
