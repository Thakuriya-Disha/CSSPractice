CSS Interview Preparation - DAY I

1) Explain inline, internal and external stylesheet.

Inline: It is defined using the "style" attribute inside the opening tag, in the HTML dcocument. Syntax: style = "property:value"

Internal: It is defined inside <head> of HTML document, using the <style> tag.

Syntax:
            <head>
                <style>
                selector {
                    property: value;
                    property: value;
                }
                </style>
            </head>

External: The self-closing <link> tag is used inside <head> with "href" attribute (hypertext reference) that links external CSS file to the HTML file, and the "rel" attribute states the relation of the linked file with the HTML file which is "stylesheet".

2) What is CSS Box Model?

The CSS Box Model is a representational model that consists of: the content area, padding, border and margin. It is a box that wraps around every HTML element. Box Model plays an important role in design and layout of a website.
Content: This is the area where the actual content (text, images, videos, forms, tables) appears.
Padding: The transparent area which is around the content.
Border: It goes around the padding and content.
Margin: It is the transparent area outside the border.

3) What is the difference between padding and margin?

PADDING:

  1. Padding is the space between content and border.
  2. Value for padding can never be negative.
  3. When value of padding is changed, the size of the element gets affected.

MARGIN:
  1. Margin is the space between border and nearby element.
  2. Value for padding can be negative.
  3. When value of margin is changed, the size of the element remains unaffected.

4) Difference between flex row and column

They are values for the flex-direction property.

  • flex-direction: row;
    • It arranges items inside a flexbox horizontally.

  • flex-direction: column;
    • It arranges items inside a flexbox vertically.

5) What does justify-content allow you to do?

Justify-content is one of the sub-properties of flexbox. It spcifies how flex items are distributed horizontally inside the flex-container. It works on the origin (also called the main axis or the X-axis).
There are 7 values of the justify-content. They are: stretch, center, flex-start, flex-end, space-between, space-around, space-evenly.

6) Difference between absolute and relative positioning

RELATIVE: It places an element relative to its current position without changing the layout around it. (that is, it leaves behind a gap)

ABSOLUTE: It places an element relative to its parent's position and changes the layout around it. (that is, we see no gap being created)

7) What is the z-index in CSS?

The z-index determines the stack level of an element. (Stack level: refers to the element's position on the z-axis) Negative values can be given to the z-index.
